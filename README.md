# Currency-converter

## Description
This program is a simple Python-based currency converter. It accepts an amount
entered in Nigerian Naira (NGN) and converts it into three foreign currencies:
United States Dollar (USD), Euro (EUR), and British Pound Sterling (GBP).

The program uses a dictionary to store exchange rates and a reusable function
to perform the conversions.

## How the Program Works
1. The user is prompted to enter an amount in Nigerian Naira (NGN).
2. The program uses predefined exchange rates stored in a dictionary.
3. The amount is converted into USD, EUR, and GBP.
4. The converted values are displayed in a clear and formatted output.

## Exchange Rates Used
The following fixed exchange rates are used in this program:
- 1 NGN = 0.00067 USD
- 1 NGN = 0.00062 EUR
- 1 NGN = 0.00054 GBP

(Note: These are sample rates for learning purposes and are not live rates.)

## Requirements
- Python 3.x

## How to Run the Program
1. Open a terminal or Python IDE.
2. Navigate to the folder containing the file.
3. Run the program using:
   python currency_converter.py
4. Enter an amount in NGN when prompted.
